## POST /api/auth/register

Creates a new user account with the provided credentials.

<Note>
  This endpoint does not require authentication.
</Note>

### Request Body

<ParamField path="username" type="string" required>
  Username for the account. Must be unique and cannot exceed 50 characters.
</ParamField>

<ParamField path="email" type="string" required>
  User's email address. Must be unique and valid email format. Maximum 255 characters.
</ParamField>

<ParamField path="password" type="string" required>
  Password for the account. Will be securely hashed before storage.
</ParamField>

<ParamField path="full_name" type="string">
  User's full name (optional). Maximum 100 characters.
</ParamField>

### Response

<ResponseField name="user_id" type="long">
  Unique identifier for the created user
</ResponseField>

<ResponseField name="username" type="string">
  Username of the created user
</ResponseField>

<ResponseField name="email" type="string">
  Email address of the created user
</ResponseField>

<ResponseField name="full_name" type="string" nullable>
  Full name of the user (if provided)
</ResponseField>

<ResponseField name="avatar_url" type="string" nullable>
  Avatar URL (null for new users)
</ResponseField>

<ResponseField name="is_active" type="boolean">
  Whether the user account is active (default: true)
</ResponseField>

<ResponseField name="email_verified" type="boolean">
  Whether the email has been verified (default: false)
</ResponseField>

<ResponseField name="created_at" type="datetime">
  Timestamp when the user was created
</ResponseField>

### Request Example

<CodeGroup>
```bash cURL
curl -X POST https://api.taskpro.com/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "username": "johndoe",
    "email": "john.doe@example.com",
    "password": "SecurePass123!",
    "full_name": "John Doe"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.taskpro.com/api/auth/register', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    username: 'johndoe',
    email: 'john.doe@example.com',
    password: 'SecurePass123!',
    full_name: 'John Doe'
  })
});

const data = await response.json();
```

```python Python
import requests

response = requests.post(
    'https://api.taskpro.com/api/auth/register',
    json={
        'username': 'johndoe',
        'email': 'john.doe@example.com',
        'password': 'SecurePass123!',
        'full_name': 'John Doe'
    }
)

data = response.json()
```
</CodeGroup>

### Response Example

<CodeGroup>
```json 201 - Success
{
  "user_id": 123,
  "username": "johndoe",
  "email": "john.doe@example.com",
  "full_name": "John Doe",
  "avatar_url": null,
  "is_active": true,
  "email_verified": false,

```

```json 400 - Bad Request
{
  "detail": "Email already registered"
}
```

```json 400 - Validation Error
{
  "detail": "Username already exists"
}
```
</CodeGroup>

### Status Codes

| Status Code | Description |
|-------------|-------------|
| 201 | User successfully created |
| 400 | Invalid input or duplicate username/email |
| 500 | Internal server error |
