---
title: "Delete Subtask"
api: "DELETE /api/subtasks/{subtaskId}"
description: "Delete a subtask"
---

Permanently deletes a subtask. This action cannot be undone. The subtask must belong to a task owned by the authenticated user.

## Authentication

Requires authentication via session cookie or Bearer token.

## Path Parameters

<ParamField path="subtaskId" type="long" required>
  The ID of the subtask to delete
</ParamField>

## Response

Returns a 204 No Content response on successful deletion.

<RequestExample>
```bash cURL
curl -X DELETE https://api.taskpro.com/api/subtasks/456 \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```python Python
import requests

url = "https://api.taskpro.com/api/subtasks/456"
headers = {
    "Authorization": "Bearer YOUR_TOKEN"
}

response = requests.delete(url, headers=headers)
print(f"Status: {response.status_code}")
```

```javascript JavaScript
const response = await fetch('https://api.taskpro.com/api/subtasks/456', {
  method: 'DELETE',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});

console.log('Status:', response.status);
```
</RequestExample>

<ResponseExample>


