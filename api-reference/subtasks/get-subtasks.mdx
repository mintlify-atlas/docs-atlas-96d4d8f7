---
title: "Get Subtasks"
api: "GET /api/tasks/{taskId}/subtasks"
description: "Retrieve all subtasks for a specific task"
---

Retrieves a list of all subtasks associated with a parent task. Only subtasks belonging to tasks owned by the authenticated user are returned.

## Authentication

Requires authentication via session cookie or Bearer token.

## Path Parameters

<ParamField path="taskId" type="long" required>
  The ID of the parent task whose subtasks you want to retrieve
</ParamField>

## Response

Returns an array of subtask objects.

<ResponseField name="subtaskId" type="long">
  Unique identifier for the subtask
</ResponseField>

<ResponseField name="taskId" type="long">
  ID of the parent task
</ResponseField>

<ResponseField name="title" type="string">
  The subtask title
</ResponseField>

<ResponseField name="description" type="string">
  The subtask description
</ResponseField>

<ResponseField name="tags" type="array">
  Array of tag objects associated with the subtask
</ResponseField>

<ResponseField name="durationTagId" type="integer">
  ID of the duration tag indicating estimated time
</ResponseField>

<ResponseField name="status" type="string">
  Current status: `pending`, `in_progress`, or `completed`
</ResponseField>

<ResponseField name="completedAt" type="datetime">
  Timestamp when the subtask was completed (null if not completed)
</ResponseField>

<ResponseField name="displayOrder" type="integer">
  Order position for displaying the subtask
</ResponseField>

<ResponseField name="createdAt" type="datetime">
  Timestamp when the subtask was created
</ResponseField>

<ResponseField name="updatedAt" type="datetime">
  Timestamp when the subtask was last updated
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET https://api.taskpro.com/api/tasks/123/subtasks \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```python Python

url = "https://api.taskpro.com/api/tasks/123/subtasks"

}

```
