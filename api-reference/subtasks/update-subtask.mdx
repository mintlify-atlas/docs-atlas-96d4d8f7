---
title: "Update Subtask"
api: "PUT /api/subtasks/{subtaskId}"
description: "Update an existing subtask"
---

Updates the details of an existing subtask. Only the fields provided in the request body will be updated. The subtask must belong to a task owned by the authenticated user.

## Authentication

Requires authentication via session cookie or Bearer token.

## Path Parameters

<ParamField path="subtaskId" type="long" required>
  The ID of the subtask to update
</ParamField>

## Request Body

All fields are optional. Only include fields you want to update.

<ParamField body="title" type="string">
  Updated title for the subtask (max 255 characters)
</ParamField>

<ParamField body="description" type="string">
  Updated description for the subtask
</ParamField>

<ParamField body="durationTagId" type="integer">
  Updated duration tag ID
</ParamField>

<ParamField body="status" type="string">
  Updated status. Options: `pending`, `in_progress`, `completed`
</ParamField>

<ParamField body="displayOrder" type="integer">
  Updated display order position
</ParamField>

<ParamField body="tagIds" type="array">
  Updated list of tag IDs (replaces existing tags)
</ParamField>

## Response

<ResponseField name="subtaskId" type="long">
  Unique identifier for the subtask
</ResponseField>

<ResponseField name="taskId" type="long">
  ID of the parent task
</ResponseField>

<ResponseField name="title" type="string">
  The updated subtask title
</ResponseField>

<ResponseField name="description" type="string">
  The updated subtask description
</ResponseField>

<ResponseField name="tags" type="array">
  Array of tag objects associated with the subtask
</ResponseField>

<ResponseField name="durationTagId" type="integer">
  ID of the duration tag
</ResponseField>

<ResponseField name="status" type="string">
  Current status of the subtask
</ResponseField>

<ResponseField name="completedAt" type="datetime">
  Timestamp when the subtask was completed (automatically set when status changes to completed)
</ResponseField>

<ResponseField name="displayOrder" type="integer">
  Display order position
</ResponseField>

<ResponseField name="createdAt" type="datetime">
  Timestamp when the subtask was created
</ResponseField>

<ResponseField name="updatedAt" type="datetime">
  Timestamp when the subtask was last updated
</ResponseField>

<RequestExample>
```bash cURL
curl -X PUT https://api.taskpro.com/api/subtasks/456 \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "status": "completed",
    "description": "Completed research on REST vs GraphQL. REST chosen for simplicity."
  }'
```

```python Python
import requests

url = "https://api.taskpro.com/api/subtasks/456"
headers = {
    "Authorization": "Bearer YOUR_TOKEN",
    "Content-Type": "application/json"
}
payload = {
    "status": "completed",
    "description": "Completed research on REST vs GraphQL. REST chosen for simplicity."
}

response = requests.put(url, json=payload, headers=headers)
print(response.json())
```

```javascript JavaScript

```
