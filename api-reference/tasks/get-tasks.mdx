## GET /api/tasks

Retrieves all tasks for the authenticated user with optional filtering.

### Authentication

Required. Use JWT Bearer token in the Authorization header.

### Query Parameters

<ParamField query="folderId" type="integer">
  Filter tasks by folder ID. If provided, only tasks in the specified folder are returned.
</ParamField>

<ParamField query="status" type="string">
  Filter tasks by status. Valid values: `pending`, `in_progress`, `completed`, `cancelled`
</ParamField>

<ParamField query="includeArchived" type="boolean" default="false">
  Whether to include archived tasks in the results
</ParamField>

### Response

Returns an array of task objects with progress information.

<ResponseField name="taskId" type="integer">
  Unique identifier for the task
</ResponseField>

<ResponseField name="userId" type="integer">
  ID of the user who owns the task
</ResponseField>

<ResponseField name="folderId" type="integer">
  ID of the folder containing the task
</ResponseField>

<ResponseField name="tags" type="array">
  Array of tag objects associated with the task
</ResponseField>

<ResponseField name="title" type="string">
  The task title
</ResponseField>

<ResponseField name="description" type="string">
  The task description
</ResponseField>

<ResponseField name="durationTagId" type="integer">
  ID of the duration tag
</ResponseField>

<ResponseField name="priority" type="integer">
  Priority level of the task
</ResponseField>

<ResponseField name="status" type="string">
  Current status: `pending`, `in_progress`, `completed`, or `cancelled`
</ResponseField>

<ResponseField name="dueDate" type="string">
  Due date in ISO 8601 format
</ResponseField>

<ResponseField name="dueTime" type="string">
  Due time in ISO 8601 format
</ResponseField>

<ResponseField name="completedAt" type="string">
  Timestamp when the task was completed (null if not completed)
</ResponseField>

<ResponseField name="displayOrder" type="integer">
  Display order in lists
</ResponseField>

<ResponseField name="isArchived" type="boolean">
  Whether the task is archived
</ResponseField>

<ResponseField name="createdAt" type="string">
  Timestamp when the task was created
</ResponseField>

<ResponseField name="updatedAt" type="string">
  Timestamp when the task was last updated
</ResponseField>

<ResponseField name="totalSubtasks" type="integer">
  Total number of subtasks
</ResponseField>

<ResponseField name="completedSubtasks" type="integer">
  Number of completed subtasks
</ResponseField>

<ResponseField name="completionPercentage" type="number">
  Percentage of subtasks completed (0-100)
</ResponseField>

### Example Request

```bash
curl -X GET "https://api.taskpro.com/api/tasks?folderId=5&status=pending" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

### Example Request (All Tasks)

```bash
curl -X GET https://api.taskpro.com/api/tasks \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

### Example Request (Include Archived)

```bash
curl -X GET "https://api.taskpro.com/api/tasks?includeArchived=true" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```
