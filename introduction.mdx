## Welcome to Task Pro API

Task Pro API is a comprehensive REST API for task management, built with Spring Boot and Kotlin. It provides a robust backend solution for building task management applications with support for folders, tags, subtasks, and more.

## Tech Stack

Task Pro API is built with modern, production-ready technologies:

- **Spring Boot 4.0.3** - Enterprise Java framework
- **Kotlin 2.2.21** - Modern, concise programming language
- **PostgreSQL** - Powerful relational database
- **JWT (Auth0)** - Secure token-based authentication
- **Docker** - Containerization for easy deployment
- **Gradle** - Build automation and dependency management
